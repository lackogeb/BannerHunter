# Security Policy

## Supported Versions
This repository is a small utility project. Security fixes are applied on a best-effort basis to the latest version on the default branch.

## Reporting a Vulnerability
If you believe you've found a security vulnerability:
1. **Do not** open a public issue with exploit details.
2. Send a report describing the issue, impact, and reproduction steps.
3. If possible, include the platform (Linux/Termux), compiler version, and how you built the binary.

## Responsible Use
BannerHunter is intended for **authorized** testing and troubleshooting only. The tool includes a required runtime confirmation flag (`--i-have-permission`) to discourage misuse.
